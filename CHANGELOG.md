# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-29

### Added
- Interactive Home Screen with Sanji's "serving" animation.
- Intelligent Recipe Search with category filtering.
- Professional Cook Mode with integrated timers and ingredient checklist.
- Kitchen Management (Smart Grocery List, Favorites, Custom Recipes).
- AI Chef Companion with dynamic greetings and context-aware responses.
- Vision capabilities to recognize ingredients from images.
- Unit test structure for ViewModels and Repositories.
- CI/CD workflow with GitHub Actions.
- Linting checks with detekt and ktlint.
- Enhanced documentation (LICENSE, CONTRIBUTING, README updates).
- Semantic versioning setup.
